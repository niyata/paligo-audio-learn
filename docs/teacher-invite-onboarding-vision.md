## Pain point

นักเรียนเข้า Paligo **อยากเรียนบาลี** แต่ **ยังไม่มีอาจารย์ตรวจข้อสอบ** → ส่งตรวจไม่ได้ · ทิ้งแพลตฟอร์ม

## วิสัยทัศน์

ผู้เรียนทุกคนมีทางไปถึงผู้ตรวจที่ไว้ใจได้ — ผ่านเครือข่ายสถาบัน หรือแชร์คำเชิญให้ครูประจำ — และส่งข้อสอบได้เร็วหลังสมัคร

## แนวทางแก้ (สองทาง)

| ทาง | สรุป |
|-----|------|
| **A — Supply** | Seed ครูล่วงหน้าในระบบ · นักเรียนเลือกจากรายการ |
| **B — Demand** | นักเรียนสร้าง invite link → แชร์ **LINE Flex + shareTargetPicker** → ครูหลายคนรับพร้อมกันได้ |

## Flow สำคัญ

1. ครูเปิดลิงก์ → **invite wizard** → **รับตัวเป็นศิษย์**
2. นักเรียนได้ **inbox** แจ้งว่ามีใครรับ (1 ท่าน / หลายท่าน)
3. ตอนส่งตรวจ **เลือกผู้ตรวจ** เพื่อแบ่งภาระ
