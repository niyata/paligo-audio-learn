## KB_06 วากยสัมพันธ์ — สรุป scope

- ครอบคลุมปฐมา–สัตตมี + วิเศษ (วิกติ, วิเสสนะ, สัมภาวนะ ฯลฯ)
- **ลำดับการแปล ๙ ประการ:** อาลปนะ → นิบาต → กาลสัตตมี → บทประธาน → บทขยายประธาน → กริยาในระหว่าง → บทเนื่องด้วยกริยาในระหว่าง → กริยาคุมพากย์ → บทเนื่องด้วยกริยาคุมพากย์
- **กฎ:** ประธานและกริยาคุมพากย์ ถ้าไม่มีต้องเติม (อัธยาหาร)
- **Anaphora:** โส(ชโน) — โส สา ตํ เต ตา ตานิ อยํ อิมํ เอโส เอสา
- **เลขนอก-เลขใน:** ห้ามปนกัน
